Authentication Routes
Authentication
POST /api/v1/signup Create user ▼

Registers a new user account in the CampCart database.

Request Body Fields
FieldTypeRequirementDescription
email string Required Valid email address
password string Required Secure password
fullName string Required Full name
Request Payload Example
{
  "email": "user@example.com",
  "password": "securePass123!",
  "fullName": "John Doe"
}
Success Response Example
{
  "message": "Account created successfully",
  "role": "user"
}
POST /api/v1/login Login & get tokens ▼

Authenticates credentials and establishes a session.

Request Body Fields
FieldTypeRequirementDescription
email string Required Registered email
password string Required Password
Request Payload Example
{
  "email": "user@example.com",
  "password": "securePass123!"
}
Success Response Example
{
  "message": "Logged in successfully",
  "userRole": "user",
  "accessToken": "ey..."
}
POST /api/v1/auth/logout Logout user ▼

Clears the access and refresh tokens from cookies.

Success Response Example
{
  "message": "Logged out successfully"
}
POST /api/v1/auth/forgot-password Forgot password ▼

Sends a password reset token to the user email.

Request Body Fields
FieldTypeRequirementDescription
email string Required Registered email
Request Payload Example
{
  "email": "user@example.com"
}
Success Response Example
{
  "message": "Password reset token generated"
}

Request Body Fields

POST SECURE /api/v1/orders Create order ▼

Creates a new order from user items.

Request Body Fields
FieldTypeRequirementDescription
items array Required Array of item objects
deliveryType string Required Either 'pickup' or 'delivery'
paymentMethod string Required Payment method (e.g. 'Credit Card')
deliveryAddress string Optional Required if deliveryType is 'delivery'
Request Payload Example
{
  "items": [{ "product": "id", "quantity": 1 }],
  "deliveryType": "delivery",
  "paymentMethod": "Credit Card",
  "deliveryAddress": "123 Main St"
}
Success Response Example
{
  "message": "Order created successfully",
  "order": { ... }
}
